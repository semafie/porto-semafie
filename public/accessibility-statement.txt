# Accessibility Statement - Restu Imam Safii Portfolio
# Website: https://semafie.upylon.com

## Commitment to Accessibility

Restu Imam Safii Portfolio is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Conformance Status

This website strives to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. These guidelines explain how to make web content more accessible for people with disabilities and more user-friendly for everyone.

## Accessibility Features

### Keyboard Navigation
- **Full Keyboard Access**: All functionality accessible via keyboard
- **Skip Navigation**: Skip to main content links
- **Focus Indicators**: Clear focus indicators for all interactive elements
- **Tab Order**: Logical tab order throughout the site

### Screen Reader Support
- **Semantic HTML**: Proper heading structure and landmarks
- **ARIA Labels**: Descriptive labels for interactive elements
- **Alt Text**: Descriptive alt text for all images
- **Form Labels**: Properly labeled form fields

### Visual Accessibility
- **Color Contrast**: High contrast ratios for text readability
- **Font Sizing**: Scalable text that can be enlarged
- **Responsive Design**: Adapts to different screen sizes
- **Dark Mode**: High contrast dark theme available

### Content Accessibility
- **Plain Language**: Clear and simple language
- **Structured Content**: Logical content organization
- **Multimedia**: Captions and transcripts where applicable
- **Links**: Descriptive link text

## Known Limitations

While we strive for comprehensive accessibility, some areas may have limitations:

- **Third-party Content**: External content may not be fully accessible
- **Complex Interactions**: Some advanced features may require alternative methods
- **Legacy Content**: Older content may not meet current standards
